# 云游博物馆（Cloud Museum Explorer）

**网址**：https://cloud-museum-qttl.vercel.app/

> 一个基于 **Vue3 + Vite + Node.js** 的在线博物馆资源聚合平台，通过统一的前端界面整合各个线上博物馆的展览资源，让用户能够在网页端实现便捷、沉浸式的线上参观体验。

## 🌟 项目简介
云游博物馆网页旨在整合多个线上博物馆的资源，集中展示各类展厅、展品信息、VR 数字内容、闯关问答以及解答疑问的AI机器人。

通过访问本网站，用户可以通过这个平台快速跳转到各大博物馆的线上展览，提升线上参观的便利性。

本项目为前端主导，使用 Vue3 打造现代化 UI，并通过 Node.js 提供必要的数据接口或资源代理。

---

## 🌟 功能介绍
- **首页**：包含精美的入场动画以及各个分页面的快捷按钮，通过最上方导航栏也可快速进行页面的切换。

- **博物馆目录**：集成了各个线上博物馆及云展馆的访问入口。通过卡片形式呈现博物馆名称及相关信息，保证了用户可以方便访问各个博物馆。

- **AI助手**：平台内嵌智能问答助手，用户可通过点击图标唤出对话窗口，对参观过程中遇到的问题或者历史知识进行即时咨询，提升系统的辅助讲解能力。

- **文物问答**：答题设有闯关模式和学习模式，用户可以根据需求自由选择，学习历史知识，闯关后可以获得文物卡片作为奖励。

- **VR博物馆**：集成了国内外博物馆VR展厅，内置小导航栏可以对博物馆进行搜索，导航栏下有各个标签，点击后可以对指定类型的博物馆进行筛选。

- **文物介绍**：平台实现了著名历史文物的集中展示与分类浏览，每个图片下方都附有介绍，用户可以通过标签筛选的方式高效查看不同时代的文物内容。

- **小屏适配**：系统支持手机、平板等小屏设备访问，页面布局会根据屏幕尺寸自动调整，保证在不同设备上的显示效果和基本交互功能。
